# Hooks

Comandos shell que se ejecutan automáticamente en puntos del ciclo de Claude Code.

## Temas
- Eventos disponibles (PreToolUse, PostToolUse, SessionStart, etc.)
- Configuración en settings.json
- Exit codes y control de decisiones
- Hooks basados en prompt (LLM)
