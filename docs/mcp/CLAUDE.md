# MCP (Model Context Protocol)

Protocolo para conectar Claude Code con herramientas y servicios externos.

## Temas
- Qué es MCP y cómo funciona
- Configurar MCP servers (`claude mcp add`)
- Servers populares (filesystem, github, databases)
- Crear MCP servers personalizados
