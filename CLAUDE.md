## Project Purpose

Repositorio personal de aprendizaje sobre Claude Code. Documenta hooks, skills, MCP, agents, workflows y best practices.

## Key Files

- **APRENDIZAJE-COMPLETO.md** - Consolidado con todo el conocimiento (referencia principal)
